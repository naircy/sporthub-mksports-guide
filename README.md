# sporthub-mksports-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以运动与体育相关内容为主，涵盖信息展示、工具使用、数据参考等多种用途。仓库本身不针对任何特定域名或网站，所有页面均为独立文件，便于浏览和分发。

## 目录说明

```
sporthub-mksports-guide/
├── index.html          # 仓库首页或导航页
├── pages/              # 存放主要 HTML 页面文件
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本说明文件
```

- **pages/**：存放所有独立 HTML 页面，每个页面功能独立。
- **assets/**：存放页面引用的公共资源，按类型分类。
- **index.html**：可作为入口或索引，方便快速浏览所有页面。

## 页面归档说明

所有 HTML 页面均为静态文件，无后端依赖，可直接在浏览器中打开运行。页面内容以运动与体育信息为主，不涉及具体商业推广或域名跳转。归档目的为方便查阅、测试或作为参考资料。

- 每个页面均包含完整 HTML 结构，部分页面可能引用外部资源（如 CDN 样式库），但无恶意链接。
- 页面标题和内容说明请参见页面内部注释或 `index.html` 中的导航链接。
